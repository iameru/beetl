# beetl

beetl is supposed to be a small sideproject

It is a tool for groups which have to get a __fixed Sum of money__ and want to __split the costs unevenly__, f.e. because the nurse has less income than the banker.

this will be a most basic implementation with a fixed calculating algorithm and no database. In case I enjoy it or get paid I'll implement stuff like chosing from different algorithms, save states to database and make a shareable link to colaborate and enable people to do their bidding themselves and more stuff.
